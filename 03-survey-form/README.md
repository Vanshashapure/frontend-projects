# Survey / Contact Form

A fully styled multi-field form with all major HTML input types and custom CSS styling for every input state.

## Tech Used
HTML | CSS | CSS Variables | Google Fonts

## What I Built
- Text, email, password, number inputs
- Dropdown select, checkboxes, radio buttons, and textarea
- Styled default, hover, and focus states for every input
- Dark theme using CSS custom properties (variables)
