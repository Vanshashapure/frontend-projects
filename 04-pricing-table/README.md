# Pricing Table

A 3-column pricing table with Free, Pro, and Enterprise plans. The Pro plan is highlighted as Most Popular.

## Tech Used
HTML | CSS | CSS Grid | Media Queries

## What I Built
- 3-column layout using CSS Grid
- Highlighted Pro card with badge, border, and scale effect
- Fully responsive — stacks to single column on mobile using media queries
- Consistent card design with hover effects on buttons
