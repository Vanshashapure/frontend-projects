# Product Landing Page

A fully responsive product landing page with hero section, features grid, testimonials, and a sticky CTA button.

## Tech Used
HTML | CSS | Flexbox | CSS Grid | Animations

## What I Built
- Full page layout with hero, features, testimonials, and footer sections
- Sticky call-to-action button using position sticky
- CSS animations on page load
- Fully responsive across desktop, tablet, and mobile
