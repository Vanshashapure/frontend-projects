# Photo Gallery with Lightbox

A responsive photo grid where clicking any image opens it in a fullscreen lightbox overlay.

## Tech Used
HTML | CSS Grid | JavaScript

## What I Built
- 3-column responsive photo grid using CSS Grid
- Hover zoom effect on images using CSS transform
- Custom lightbox overlay built with pure JavaScript
- Close lightbox via close button or clicking dark background
- Responsive — 2 columns on tablet, 1 column on mobile
