# Analytics Dashboard — Capstone Project

A fully responsive analytics dashboard with sidebar navigation, stat cards, a live chart, and a data table.

## Tech Used
HTML | CSS Grid | JavaScript | Chart.js

## What I Built
- Responsive sidebar navigation with active link highlighting
- Summary stat cards showing key metrics
- Bar chart built using Chart.js library
- Data table with clean row styling
- Fully responsive layout — sidebar collapses on mobile
- Deployed live on GitHub Pages
