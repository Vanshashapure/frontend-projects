# Interactive Quiz App

A 5-question quiz application with dynamic rendering, answer validation, score tracking, and a results screen.

## Tech Used
HTML | CSS | JavaScript | DOM Manipulation

## What I Built
- 5 questions stored as array of objects in JavaScript
- Dynamic question and option rendering using createElement and appendChild
- Correct answer highlights green, wrong answer highlights red
- Score tracked across all questions with final result screen
- Restart button resets all state and replays the quiz
- Built with zero libraries — pure vanilla JavaScript
